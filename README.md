# 🏭 Warehouse Inventory System (Mini Project)

## 📌 Project Description

The **Warehouse Inventory System** is a simple Python-based application designed to manage inventory efficiently. It allows users to add, update, delete, and view items in a warehouse using a menu-driven interface.

This project is ideal for beginners to understand basic Python concepts like dictionaries, functions, and loops.

---

## 🎯 Objectives

* To develop a basic inventory management system
* To understand CRUD operations (Create, Read, Update, Delete)
* To practice Python programming using functions and loops
* To implement a user-friendly menu-driven program

---

## ⚙️ Features

* ➕ Add new items with quantity
* ✏️ Update existing item quantity
* ❌ Delete items from inventory
* 📋 Display all inventory items
* 🔄 Continuous menu until user exits

---

## 🛠️ Technologies Used

* Python 3

## ▶️ How to Run the Project

1. Install Python (if not already installed)
2. Download or clone this repository
3. Open terminal or command prompt
4. Run the program:

```
python warehouse_inventory.py
```

---

## 📖 How It Works

* The inventory is stored using a Python dictionary
* Users interact with the system via a menu
* Based on user input, corresponding functions are executed
